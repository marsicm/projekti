1.1

Analiza wireframe-a

10 min

Svaku vizualnu sekciju wireframe-a pretvaramo u HTML tag. Identificiraj 4 glavna dijela:

<nav> — Logo lijevo + linkovi desno
<header> — Hero: naslov, opis, gumb
<main> — Glavni sadržaj:
<section class="usluge"> — 3 kartice
<section class="recenzije"> — 2 kolone
<section class="o-nama"> — tekst + mapa
<footer> — Kontakt forma

Semantički tagovi HTML5

Umjesto da sve stavljamo u <div>, koristimo tagove koji opisuju značenje sadržaja. Prednosti: bolja čitljivost koda, pristupačnost, SEO.

<nav>blok navigacije
<header>zaglavlje stranice ili sekcije
<main>glavni, jedinstveni sadržaj stranice
<section>tematski grupiran sadržaj
<article>samostalan sadržaj (kartica, post)
<footer>podnožje stranice ili sekcije

SAT 2 — CSS Reset i Navigacija

Trajanje: 40 minuta • Tema: Grid i Flexbox osnove

CSS Reset • display: grid • display: flex
2.1

CSS Reset i box-sizing

8 min

Otvori style.css. Na samom početku dodaj reset koji uklanja razlike između browsera:

CSS
* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

body {
    font-family: Arial, sans-serif;
    color: #333;
}

Zašto box-sizing: border-box?

Bez toga, padding se dodaje na definiranu širinu, što otežava izračun. S border-box padding je uključen u širinu — puno predvidljivije!

content-box (zadano)
width: 200px + padding: 20px
= 240px ukupno
border-box
width: 200px (padding uključen)
= 200px ukupno
2.2

Navigacija — CSS Grid

15 min

Wireframe: logo lijevo, linkovi desno. Dvije kolone → koristimo Grid.

display: grid
150px
LOGO
1fr (ostatak prostora)
LINKOVI
grid-template-columns: 150px 1fr

Što znači 1fr?

fr je fraction (razmjerni dio). Kolona s 1fr zauzima sav preostali slobodni prostor nakon što se dodijeli fiksna kolona.

CSS
/* ===== NAVIGACIJA ===== */
.navigacija {
    display: grid;
    grid-template-columns: 150px 1fr;
    align-items: center;
    padding: 20px 40px;
    border-bottom: 1px solid #ddd;
}

.logo {
    font-size: 1.1rem;
    font-weight: bold;
    letter-spacing: 2px;
}

Spremi i osvježi browser — logo i lista se sada raspoređuju u 2 stupca.

2.3

Navigacijski linkovi — Flexbox

17 min

Linkovi trebaju biti u redu i gurnuti na desnu stranu. To je posao za Flexbox s justify-content: flex-end.

display: flex  •  justify-content: flex-end
← slobodan prostor
About
Services
Contact
Contact

Osi u Flexboxu

justify-contentraspored po glavnoj osi (default: horizontalnoj)
align-itemsraspored po sporednoj osi (vertikalno)
flex-direction: columnmijenja glavnu os na vertikalnu
CSS

.nav-linkovi {
    display: flex;
    justify-content: flex-end;
    align-items: center;
    gap: 30px;
    list-style: none;
}

.nav-linkovi a {
    text-decoration: none;
    color: #333;
    font-size: 0.9rem;
}

.nav-linkovi a:hover {
    opacity: 0.6;
}

/* Gumb (obrub) u navigaciji */
.nav-linkovi .gumb {
    border: 1px solid #333;
    padding: 7px 18px;
}

.nav-linkovi .gumb:hover {
    background: #333;
    color: white;
    opacity: 1;
}

SAT 5 — Responzivni dizajn i Media Queries

Trajanje: 40 minuta • Tema: @media, breakpoints, responzivna navigacija

@media • max-width • flex-wrap
5.1

Što je responzivni dizajn?

10 min

Responzivna stranica prilagođava izgled prema širini zaslona uređaja. Isti HTML, ali CSS mijenja raspored ovisno o veličini prozora.

Viewport meta tag — zašto je bitan?

Već ga imamo u <head>:

HTML
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Bez ovog taga, mobilni browser bi skalirao stranicu kao da je široka 980px. S njim, stranica se prikazuje u stvarnoj širini zaslona.

🔍 Testiranje u DevToolsu

  1. Pritisni F12 u Chromeu
  2. Klikni ikonu mobitela (Toggle device toolbar)
  3. Odaberi uređaj ili unesi širinu ručno (npr. 375px za iPhone)
  4. Osvježi stranicu i promatraj promjene

Naši breakpoints

Breakpoint je širina zaslona na kojoj mijenjamo CSS. Koristit ćemo dva:

NazivBreakpointUređaji
Desktop bez ograničenja Računala, veći laptopi
Tablet max-width: 768px Tableti, manji laptopi
Mobitel max-width: 480px Pametni telefoni
ⓘ Desktop-first pristup: Prvo pišemo CSS za desktop (što smo radili do sad), a zatim dodajemo @media pravila koja prepisuju stilove za manje zaslone.
5.2

Sintaksa media queries

12 min

Media query se piše s @media pravilom. CSS unutar vitičastih zagrada primjenjuje se samo kad je uvjet ispunjen.

Kako čitati media query?

@media (max-width: 768px) čita se kao:
"Primijeni ove stilove kad je širina prozora manja ili jednaka 768px"

Na dnu style.css dodaj oba bloka (redosljed je bitan — tablet dolazi prvi):

CSS
/* ===== RESPONZIVNOST ===== */

/* ---- Tablet i manji (≤ 768px) ---- */
@media (max-width: 768px) {

    /* Ovdje pišemo CSS koji mijenja
       izgled na tabletima */

}

/* ---- Mobitel (≤ 480px) ---- */
@media (max-width: 480px) {

    /* Ovdje pišemo CSS koji mijenja
       izgled na mobitelima */

}
Bitan redoslijed! 480px blok mora doći ispod 768px bloka — inače se mobilni stilovi neće primijeniti.
5.3

Responzivna navigacija

18 min

Na desktopu navigacija ima 2 Grid kolone. Na tabletu logo i linkovi ne stanu u red — prelazimo na 1 kolonu, a linkove centriramo Flexboxom.

Desktop — grid-template-columns: 150px 1fr
LOGO
About   Services   Contact   [Contact]
Tablet/Mobitel — grid-template-columns: 1fr
LOGO
About
Services
Contact

justify-content: center • flex-wrap: wrap

Dodaj unutar @media (max-width: 768px) bloka:

CSS
@media (max-width: 768px) {

    /* Navigacija: 1 kolona umjesto 2 */
    .navigacija {
        grid-template-columns: 1fr;
        padding: 15px 20px;
        text-align: center;
    }

    /* Linkovi: centriranje i prelom u novi red */
    .nav-linkovi {
        justify-content: center;
        flex-wrap: wrap;
        gap: 10px;
        margin-top: 10px;

}

Što radi flex-wrap: wrap?

Na malim zaslonima, ako nema mjesta za sve linkove u jednom redu, Flexbox ih automatski prelama u sljedeći red. Bez flex-wrap, svi ostaju u jednom redu i izlaze van ekrana.

SAT 6 — Responzivni Grid i završno testiranje

Trajanje: 40 minuta • Tema: auto-fit, minmax(), tipografija, DevTools

auto-fit • minmax() • clamp()
6.1

Responzivna hero sekcija i tipografija

8 min

Na malim zaslonima naslovi su preveliki, a sekcija previsoka. Smanjujemo font-size i padding:

CSS
@media (max-width: 768px) {

    /* (Ovdje su već navigacijski stilovi) */

    .hero {
        min-height: 300px;
        padding: 40px 20px;
    }

    .hero h1 {
        font-size: 2.5rem;
        letter-spacing: 2px;
    }

    .usluge { padding: 50px 20px; }
    .o-nama  { padding: 50px 20px; }
}

@media (max-width: 480px) {

    .hero h1 { font-size: 2rem; }

}

Napredni tip: clamp()

Umjesto media queryja samo za font, možeš koristiti clamp() koji automatski skalira vrijednost:

CSS
.hero h1 {
    /* min, preferred, max */
    font-size: clamp(2rem, 5vw, 3.5rem);
}

Čita se: "font ne manji od 2rem, ne veći od 3.5rem, idealno 5% širine prozora". Nema potrebe za media queryjem!

6.2

Responzivne kartice — auto-fit i minmax()

15 min

Imamo dvije mogućnosti za responzivne kartice:

Pristup A: Media queries (ručno)

CSS
/* Desktop: 3 kolone */
.usluge-grid {
    grid-template-columns: repeat(3, 1fr);
}

/* Tablet: 2 kolone */
@media (max-width: 768px) {
    .usluge-grid {
        grid-template-columns: repeat(2, 1fr);
    }
}

/* Mobitel: 1 kolona */
@media (max-width: 480px) {
    .usluge-grid {
        grid-template-columns: 1fr;
    }
}

→ Funkcionira, ali mora se pisati za svaki breakpoint.

★ Pristup B: auto-fit + minmax() (elegantniji)

CSS
/* Desktop, tablet i mobitel — automatski! */
.usluge-grid {
    grid-template-columns:
        repeat(auto-fit, minmax(250px, 1fr));
}

→ Nema media querija. Grid sam odlučuje koliko kolona stane.

Kako radi repeat(auto-fit, minmax(250px, 1fr))?

auto-fitGradi onoliko kolona koliko stane
minmax(250px, 1fr)Svaka kolona: min 250px, max sav slobodni prostor

Na zaslonu 900px → 3 kolone × 250px = 750px → stanu 3.
Na zaslonu 550px → 2 kolone × 250px = 500px → stanu 2.
Na zaslonu 300px → 1 kolona.

U ovom tutorijalu koristimo Pristup A (media queries) jer je korak-po-korak pregledniji za učenje. Pristup B je elegantniji i preporuča se u praksi.
