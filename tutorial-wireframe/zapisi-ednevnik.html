SAT 1 Analiza wireframe-a i HTML struktura web stranice 45 min
Nastavna jedinica: Kreiranje web stranice po wireframe-u – HTML struktura
Opis aktivnosti: Učenici su analizirali wireframe predložak landing stranice i identificirali semantičke HTML elemente. Primijenili su HTML5 semantičke tagove (nav, header, main, section, article, footer) za izgradnju strukturne osnove web stranice. Kreirana je kompletna HTML datoteka s ispravnom hijerarhijom elemenata.
Nastavne metode: Demonstracija, individualni praktični rad, razgovor
Nastavna sredstva: VS Code, web browser, wireframe predložak, tutorial dokument
Odgojno-obrazovni ishodi:
  • Učenik analizira wireframe i prepoznaje odgovarajuće HTML tagove za svaku sekciju
  • Učenik koristi semantičke HTML5 elemente za strukturiranje web stranice
  • Učenik ispravno gradi hijerarhiju HTML dokumenta
SAT 2 CSS reset i pozicioniranje navigacije 45 min
Nastavna jedinica: Kreiranje web stranice po wireframe-u – CSS Grid i Flexbox u navigaciji
Opis aktivnosti: Učenici su primijenili CSS reset i svojstvo box-sizing: border-box. Pozicionirali su navigacijsku traku korištenjem CSS Grid layout-a (grid-template-columns: 150px 1fr) za raspored loga i linkova u dva stupca. Navigacijske linkove poravnali su Flexboxom (justify-content: flex-end, align-items: center).
Nastavne metode: Demonstracija, individualni praktični rad
Nastavna sredstva: VS Code, web browser, tutorial dokument
Odgojno-obrazovni ishodi:
  • Učenik primjenjuje CSS reset i objašnjava svrhu svojstva box-sizing
  • Učenik razlikuje Grid (2D) i Flexbox (1D) te odabire prikladnu metodu za zadani raspored
  • Učenik koristi display: grid i grid-template-columns za dvodimenzionalni raspored elemenata
  • Učenik primjenjuje display: flex i justify-content za poravnanje navigacijskih elemenata
SAT 3 Hero sekcija i sekcija usluga 45 min
Nastavna jedinica: Kreiranje web stranice po wireframe-u – Flexbox centriranje i Grid s repeat()
Opis aktivnosti: Učenici su centrirali sadržaj hero sekcije horizontalno i vertikalno korištenjem Flexboxa s flex-direction: column. Implementirali su sekciju usluga s tri jednake kartice korištenjem CSS Grid funkcije repeat(3, 1fr) i mjernom jedinicom fr. Primijenjeno je svojstvo object-fit: cover za ispravno prikazivanje slika unutar kartica.
Nastavne metode: Demonstracija, individualni praktični rad
Nastavna sredstva: VS Code, web browser, tutorial dokument
Odgojno-obrazovni ishodi:
  • Učenik koristi Flexbox s flex-direction: column za vertikalno slaganje i centriranje sadržaja
  • Učenik primjenjuje grid-template-columns: repeat(3, 1fr) za jednoliku distribuciju elemenata
  • Učenik objašnjava razliku između mjerne jedinice fr i fiksnih jedinica (px, %)
  • Učenik koristi svojstvo gap za definiranje razmaka između grid elemenata
SAT 4 Završetak izrade stranice – recenzije, O nama i Footer 45 min
Nastavna jedinica: Kreiranje web stranice po wireframe-u – završetak i samostalna vježba
Opis aktivnosti: Učenici su dovršili stilizaciju preostalih sekcija stranice: recenzijama i sekcijom O nama u dvorazrednom Grid layoutu (1fr 1fr s align-items: center) te podnožjem stranice u Flexbox rasporedu. U samostalnoj vježbi učenici su personalizirali stranicu promjenom boja, fontova i dodavanjem hover efekata s transition. Provedena je usporedba s gotovim rješenjem.
Nastavne metode: Demonstracija, individualni praktični rad, samostalno rješavanje zadataka
Nastavna sredstva: VS Code, web browser, tutorial dokument, gotovo rješenje
Odgojno-obrazovni ishodi:
  • Učenik kombinira Grid i Flexbox u istoj sekciji za složene rasporede
  • Učenik koristi align-items: center na Grid kontejneru za vertikalno poravnanje stupaca
  • Učenik samostalno prilagođava vizualne elemente stranice (boje, fontovi, hover efekti)
  • Učenik izrađuje kompletnu statičnu web stranicu prema zadanom wireframe predlošku
SAT 5 Osnove responzivnog dizajna i media queries 45 min
Nastavna jedinica: Responzivni web dizajn – @media pravila i responzivna navigacija
Opis aktivnosti: Učenici su upoznali koncept responzivnog dizajna i važnost viewport meta taga. Uveden je desktop-first pristup razvoja. Primijenili su @media pravila za breakpoints 768px (tablet) i 480px (mobitel) te prilagodili navigacijsku traku za manje zaslone korištenjem flex-wrap: wrap i justify-content: center. Korišten je Chrome DevTools za simulaciju različitih uređaja.
Nastavne metode: Demonstracija, individualni praktični rad
Nastavna sredstva: VS Code, web browser, Chrome DevTools, tutorial dokument
Odgojno-obrazovni ishodi:
  • Učenik objašnjava pojam responzivnog dizajna i svrhu viewport meta taga
  • Učenik razlikuje desktop-first i mobile-first pristup razvoja
  • Učenik piše @media pravila za zadane breakpoints i ispravno ih poredava
  • Učenik koristi Chrome DevTools za testiranje izgleda stranice na različitim uređajima
  • Učenik primjenjuje flex-wrap za prelom Flexbox elemenata u novi red
SAT 6 Responzivni Grid i završno testiranje 45 min
Nastavna jedinica: Responzivni web dizajn – responzivni Grid, auto-fit, minmax() i testiranje
Opis aktivnosti: Učenici su prilagodili Grid rasporede za manje zaslone (kartice usluga: 3 → 2 → 1 stupac; dvorazredne sekcije: 2 → 1 stupac). Uspoređena su dva pristupa: ručni media queries i napredna tehnika repeat(auto-fit, minmax(250px, 1fr)). Uvedena je CSS funkcija clamp() za fluidnu tipografiju. Provedeno je završno testiranje stranice na četiri veličine zaslona u Chrome DevToolsu.
Nastavne metode: Demonstracija, individualni praktični rad, provjera rezultata
Nastavna sredstva: VS Code, web browser, Chrome DevTools, tutorial dokument, gotovo rješenje
Odgojno-obrazovni ishodi:
  • Učenik primjenjuje media queries za prilagodbu Grid rasporeda na tabletima i mobitelima
  • Učenik koristi repeat(auto-fit, minmax()) kao alternativu ručnim media queryjima za Grid
  • Učenik objašnjava prednosti funkcije clamp() za fluidnu tipografiju
  • Učenik sustavno testira responzivnost web stranice na različitim veličinama zaslona
  • Učenik izrađuje kompletnu responzivnu web stranicu prema wireframe predlošku
